# Chronicle Web

Frontend for Chronicle — Vite + React + TanStack Router + TanStack Query.

## Prerequisites

- Node.js 20+
- pnpm 11+
- The API server running at `http://localhost:8080` (see `../api/`)

## Setup

```bash
# from chronicle/web/
pnpm install
```

Most commands have a `make` shortcut at the repo root — run `make help` there.

## Commands

```bash
pnpm dev        # start dev server at http://localhost:5173
pnpm build      # production build → dist/
pnpm typecheck  # tsc --noEmit, no output = no errors
pnpm lint       # ESLint
pnpm orval      # regenerate src/api/ from the running API's OpenAPI spec
                # requires the API server to be up at localhost:8080
```

From the repo root you can also use:

```bash
make web        # same as pnpm dev from web/
make orval      # same as pnpm orval from web/
make down       # stop all dev containers
```

## Generating typed API hooks

`src/api/` is generated — never edit it by hand.

1. Start the API server: `make api` (from repo root) or `cd ../api && go run cmd/server/main.go`
2. Run: `make orval` (from repo root) or `pnpm orval` (from this directory)

This fetches `http://localhost:8080/openapi.json` and writes typed TanStack Query hooks to `src/api/index.ts`.

## Project structure

```
src/
  api/              # generated by orval — do not edit
  components/
    ui/             # Radix-based primitives
  lib/
    axios.ts        # exports apiClient (axios instance) and api (orval mutator fn)
  routes/           # TanStack Router file-based routes
    __root.tsx
    index.tsx
    login.tsx
    projects.tsx
  routeTree.gen.ts  # generated by Vite plugin on pnpm dev — do not edit
  main.tsx
  index.css
```

## Environment

The dev server proxies `/api/*` to `http://localhost:8080`. No `.env` needed for local dev.
